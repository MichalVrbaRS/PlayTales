# PlayTale Audiobook Feature Backlog

## Proposed Next Features

1. Bookmarking
- Per-book chapter bookmarks with custom names and quick jump.

2. Chapter Trim-Silence / Smart Skip
- Optional skip-silence mode for spoken audiobooks.

3. Queue / Up Next
- Manual chapter queue and play-next reordering.

4. Stats + Goals
- Listening time, streaks, daily goals, and progress dashboard.

5. Playback EQ Presets
- Voice clarity preset and simple tone controls.

6. Better Import Metadata
- Parse author/series/year from tags and file structure.

7. Search Upgrades
- Search by chapter title and recently played.

8. Pin/Favorite Books
- Quick access shelf and sorting by recent/title/progress.

9. Cloud Backup / Sync
- Backup DB + progress (optionally covers/settings) to cloud.

10. Sharing / Export
- Export progress and bookmarks as JSON/CSV.

11. Android Notification Completion
- Media-style notification actions and lock-screen artwork consistency.

12. Testing + Reliability
- Automated tests for import dedup, progress persistence, sleep timer, and seek behavior.

## Suggested First Implementation Pair

- Bookmarking
- Pin/Favorite Books
